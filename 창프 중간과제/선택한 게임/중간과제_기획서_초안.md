# 게임 기획서

## 기본정보
- 게임 제목 : 
- 장르 : 아케이드 액션
- 기반 템플릿 : dodger

## 게임 규칙
- 목표 : 일정 점수 도달 (적을 피하거나, 적을 맞출 시 점수 증가)
- 승리 조건 : 1500점 도달 (추후 조정 가능성 존재)
- 패배 조건 : 체력 모두 소진시 패배
- 조작 방법 : 방향키 or WASD 로 조작

## 차별점
- 기존에는 단순 피하기만 있었다면, 현재 수정하려는 게임에서는 방어막을 추가하여다가오는 적을 방어막으로 튕겨내, 다가오는 적을 맞추면 맞췄다는 파티클과 추가 점수를 얻을 수 있는 시스템으로 구현하려 한다. 
- 현재 초안기준으로는 상시 방어막이 가능하도록 만들었으나, 추후에 플레이어와 적이 일정 거리 이하로 가까워졌을 시 에만 발동되게 만들어 타이밍에 대한 숙련작업이 필요하도록 만들 계획이다.
- 현재 고려중인 사항은 시작 위치를 중간으로 옮겨서, 위 아래로 적을 다가오게 만들어서 더욱 숨통을 조여오는 느낌을 주려고 한다.

## 난이도 설계
- 기본적으로 있는 난이도 3단계에서 확장 할 예정이다. 지금 예정사항으로는 기본적으로 위or아래 에서 다가온다고 하면, 난이도가 올라간 후에는 양옆에서도 다가오게 만들 생각까지 하고 있다.
